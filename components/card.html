Cards

Cards serve as clickable elements that link to project details or other pages. They contain an image, title, description, and action.

Example

Project Title

A brief description of the project outlining its key features and goals. This text gives users a quick overview.

Learn More

Code

<div class="card-amber"> <div class="card-img-container"> <!-- Replace src with your image path --> <img src="images/card-image.jpg" alt="Project Image"> </div> <div class="card-body-amber"> <h4 class="card-title-amber">Card Title</h4> <p class="card-text-amber">Brief description of the project or content.</p> <a href="#" class="btn btn-pastel">Learn More</a> </div> </div>

Usage

  • Use cards to display content composed of different elements whose size or supported actions vary.
  • Cards should always be uniform in height when displayed in a grid row.
  • Ensure the "Learn More" button uses the .btn-pastel class for the correct brand color and hover effect.
  • Content within the card should be center-aligned for a balanced look.

Accessibility Considerations

Ensure that the text color contrast against the card background meets WCAG 2.1 AA standards for readability. The current design uses Charcoal text on a White background, which passes these standards.
