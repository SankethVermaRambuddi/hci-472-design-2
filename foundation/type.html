Typography

The specific font stacks and text styles that define the Amber aesthetic.

Headings

We use Roboto for all major headings. It is a geometric, modern sans-serif that creates a clean and strong visual hierarchy.

Body/Paragraphs

We use Open Sans for body text. It is a highly legible humanist sans-serif that is optimized for print, web, and mobile interfaces.
